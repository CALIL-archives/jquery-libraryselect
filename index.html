jQuery Library Select

  • Download

jQuery Library Select は、図書館選択を簡単にするjQueryプラグインです。

Getting Started

以下のコードでまず始めてみましょう！ 

<!-- Include Twitter Bootstrap and jQuery: -->
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/2.1.1/jquery.min.js"></script>
<link href="http://netdna.bootstrapcdn.com/bootstrap/3.1.1/css/bootstrap.min.css" rel="stylesheet">
<script src="http://netdna.bootstrapcdn.com/bootstrap/3.1.1/js/bootstrap.min.js"></script>

<!-- Include the plugin's JS: -->
<script src="http://calil.github.io/jquery-libraryselect/jquery-libraryselect.js"></script>

<!-- Build your input: -->
<input type="text" name="libraryselect" id="libraryselect" class="form-control input-sm" />

<!-- Initialize the plugin: -->
<script type="text/javascript">
  $(document).ready(function() {
    $('#libraryselect').libraryselect();
  });
</script>

Examples

通常のテキストボックスです。テキストボックスにキーワードを入れると図書館の候補が表示されます。
選択した図書館の図書館名を指定された場所に表示します。
ページのロード時にテキストボックスの図書館IDを元に図書館名を指定された場所に表示するサンプルです。
Gifu_Nakatsugawa X-editableを使ったサンプルです。
Kyoto_Kyoto X-editableで、選択した図書館の図書館名を指定された場所に表示するサンプルです。

Options

オプション 説明 使い方
font-size 図書館候補のフォントサイズを指定します。 
<script type="text/javascript">
  $(document).ready(function() {
    $('#libraryselect').libraryselect({
      'font-size' : '16px'
    });
  });
</script>
label 選択された図書館の図書館名を表示する場所のCSSセレクターを記述します。 
<script type="text/javascript">
  $(document).ready(function() {
    $('#libraryselect').libraryselect({
      'label' : '#sample_label'
    });
  });
</script>
keyword 既定値がある場合、このパラメーターにセットするとページのロード時に既定値を元にlabelで指定された場所に図書館名を挿入します。 
<script type="text/javascript">
  $(document).ready(function() {
    $('#libraryselect').libraryselect({
      'label'   : '#sample_label',
      'keyword' : 'Gifu_Nakatsugawa'
    });
  });
</script>
onselect 図書館が選ばれた時に呼ばれる関数です。
引数にsystem_id(図書館ID)を渡します。デフォルトでは、テキストボックスにsystem_idを入れます。		 
<script type="text/javascript">
  $(document).ready(function() {
    $('#libraryselect').libraryselect({
      'onselect' : function(system_id){
        $('.libraryselect').val(system_id)
      }
    });
  });
</script>
